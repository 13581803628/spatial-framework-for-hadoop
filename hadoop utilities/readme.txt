Provides helper utilities for :
	* Building feature sets from JSON defined in the ArcGIS Server Rest API.  More specifically, building
	  an array of features that have both attributes and geometry defined.
	  - see http://help.arcgis.com/en/arcgisserver/10.0/apis/rest/index.html?query.html#response

Dependencies :
  esri-geometry-api (http://github.com/Esri/geometry-api-java)
  jackson-core (tested with 1.9.11)
  jackson-mapper (tested with 1.9.11)
